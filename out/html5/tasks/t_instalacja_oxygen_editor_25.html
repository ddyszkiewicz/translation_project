Instalacja Oxygen Editor 25

Instalacja Oxygen XML Editor w systemie Windows odbywa się przez wykonywanie instrukcji wyświetlanych na ekranie przez dedykowany kreator instalacji.

Pobierz instalator odpowiedni dla Twojego komputera, a następnie przejdź do instalacji:

  • pobierz dla Microsoft Windows
  • Pobierz dla Mac OS
  • Pobierz dla Linux
  1. Kliknij dwukrotnie pobrany wcześniej plik o nazwie oxygen-64bit-openjdk.exe i poczekaj na uruchomienie się kreatora instalacji.

    Domyślnie wszystkie pobrane pliki znajdują się w folderze Pobrane (Downloads) wewnątrz katalogu użytkownika Windows.

  2. W oknie dialogowym wyboru języka (Language selection) upewnij się, że ustawiony jest język angielski i kliknij OK.
  3. W oknie dialogowym kontroli konta użytkownika (User Account Control) kliknij Tak aby zezwolić instalatorowi Oxygen XML Editor na dokonywanie zmian na twoim komputerze.
  4. Wykonuj polecenia wyświetlane na ekranie i potwierdzaj swoje wybory za pomocą przycisku Next.
  5. Na etapie wyboru powiązań plików (Select File Associations) możesz zostawić domyślne powiązania lub opcjonalnie dostosować wybór według swoich upodobań. Kliknij Next i dokończ instalację.
  6. Na ostatnim etapie instalacji upewnij się, że opcja Run Oxygen XML Editor jest zaznaczona i kliknij Finish.
Oxygen XML Editor 25 został poprawnie zainstalowany.

Uruchom komputer ponownie.
